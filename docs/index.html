🎵 Mica

An Experimental Systems Language
Minimal • Industrial • Composable • Auditable
Mica explores a tiny, expressive core with explicit effects, deterministic concurrency, and predictable performance. Learn the entire compiler in a weekend, extend it on Monday.
🚀 Get Started 📖 Language Tour ⭐ View on GitHub

Why Mica?

A learning-sized, industrially-inspired language playground designed for readers who want to inspect a full compiler without wading through millions of lines of code.

⚡

Tiny but Expressive

Learn the entire syntax in an afternoon. Extend behavior via standard libraries instead of bespoke keywords.

🔒

Deterministic Concurrency

Structured spawn/await with effect tracking ensures reproducible outcomes every time.

🎯

Capability-Based Effects

Explicit using clauses make side effects and resource access fully auditable.

🛠️

Ahead-of-Time Tooling

CLI exposes every compiler stage from tokens to native code, backed by golden-file snapshots.

🔌

Interop-Friendly

C-compatible calling conventions and hooks for Python/JavaScript keep the ecosystem friendly.

📚

Learning-Focused

Study real implementations of lexing, parsing, semantic analysis, lowering, and code generation.

Quick Example

Pattern matching and capability-aware logging in action

module hello capability Console func main(using Console console) -> Result[Int, Error] { let hour = 17 let message = match hour { hour if hour < 12 -> "Good morning" hour if hour < 18 -> "Good afternoon" _ -> "Good evening" } console.println(message) Ok(0) }
